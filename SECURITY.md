# Security Policy

## Supported Versions

UdaanJEE OS is pre-1.0. Security fixes are applied on the `main` branch.

## Reporting a Vulnerability

- Do not open public issues for security vulnerabilities.
- Report privately to `security@udaanjee.org` with reproduction details.
- Expected response window: 72 hours.
- Critical vulnerabilities receive expedited patching and disclosure notes.

## Scope

- Authentication and authorization bypasses
- Data exposure and PII leaks
- Remote code execution or command injection
- Multi-tenant boundary breaks
