# Feature Details (1-70)

## 1-10 Planning and Onboarding
1. Student onboarding captures class, board, target exam, study hours, and constraints.
2. Bridge readiness test evaluates class 10 fundamentals for class 11 transition.
3. Personalized roadmap generator maps monthly milestones to exam windows.
4. Unified timetable merges school, coaching, and self-study commitments.
5. Backlog auto-recovery allocates smart catch-up blocks.
6. Energy-based planning assigns high-load topics to peak focus windows.
7. Weekly rebalance adapts plans using latest performance and missed tasks.
8. Prerequisite graph ensures topic order is concept-safe.
9. Syllabus tracker supports NCERT, Main, and Advanced progression views.
10. Daily task engine prioritizes tasks using urgency and learning impact.

## 11-20 Revision, Doubt, Practice, Mocks
11. Revision scheduler uses forgetting-curve based review intervals.
12. Formula sheet builder compiles chapter formulas and use-cases.
13. Concept card generator converts weak areas into quick revision cards.
14. Doubt solving supports text and image workflow.
15. Hint ladder mode reveals guidance before final answers.
16. Peer discussions provide structured doubt collaboration.
17. Mentor rooms route unresolved doubts to subject experts.
18. Chapter-wise practice sets target depth in specific chapters.
19. Mixed-topic sets improve switching ability and retention.
20. Full mock tests simulate Main and Advanced style pressure.

## 21-30 Scoring, Analytics, and Coaching Sync
21. Negative-marking aware auto-evaluation.
22. Time-per-question analytics identifies speed bottlenecks.
23. Accuracy heatmaps detect chapter-level precision gaps.
24. Error taxonomy classifies mistake type for focused correction.
25. Weak-topic engine detects recurring low-confidence zones.
26. Intervention planner generates 7-day corrective action plans.
27. Performance dashboard visualizes score trend trajectories.
28. Percentile estimator offers low-pressure comparative benchmarking.
29. DPP tracker maps coaching assignments to completion status.
30. Missed-class catch-up planner schedules fast recovery loops.
