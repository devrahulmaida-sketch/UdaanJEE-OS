# Platform Modules

## Domain services
- Identity and access service
- Student profile and progression service
- Planner and workload optimizer
- Learning graph and content indexing service
- Assessment delivery and scoring service
- Analytics and intervention service
- Doubt resolution orchestration service
- Notification and communication service
- Mentor workspace service
- Community moderation service

## Infrastructure services
- API gateway with rate control and policy routing
- Event bus and workflow orchestration layer
- Search and semantic retrieval layer
- Blob storage for notes, doubts, and media
- Audit and compliance ledger

## Shared platform capabilities
- Feature flag framework
- Experimentation framework (A/B)
- Telemetry standardization
- Localization engine
- Accessibility compliance checks
